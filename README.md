# Timezone Clock Project

Application that shows the time based on the current timezone.

## Scripts

### Client

```
    npm install
    npm run dev
```

### Server

```
    npm install
    npm run dev
```

## Technologies

- Vue JS
- Typescript
- Node JS
- Express JS
